作成日: 2026-05-12（2026-05-14 追記: 施策8 画像SERP対応を追加）
対象サイト: https://little-seeds.ayako-suzuki.net/
データソース: Google Search Console（2026-03-06〜05-12, 3ヶ月）／GA4（2026-04-14〜05-11, 4週間）／競合検索結果（2026-05-12）

結論: 2026年3月の二軸戦略（地域×スタイル）は 半分達成、半分未達。地域軸の「藤沢」は順位8-9位で安定したが、スタイル軸（ナチュラル/自然体）と並行地域（湘南/鎌倉/茅ヶ崎）は 表示すら出ていない。一方で 検索流入比率は 14.8% → 32.9% に倍増しており、テクニカル基盤と既存トラフィック品質は良好。

次の3ヶ月は 「広げる」より「深める」 に方針転換すべき。

  1. 「ニューボーンフォト 藤沢」で2-3位獲得に資源集中（最大の伸び代）
  2. 商圏を藤沢/茅ヶ崎/鎌倉の3市に絞る（横浜・湘南は撤退、ライフパレット寡占を避ける）
  3. CV導線の強化（CV2件はすべて/contact直来訪、回遊CVゼロ）
  4. MEO（GBP口コミ獲得）を最優先施策に格上げ
  5. スタイル軸は 記事型ロングテール に切り替え、正面勝負はしない

3. 課題の構造的整理

課題A: 新設ページがSEO上の入口になっていない

GSCで gallery/plan/about/voice/faq は 合計10表示／2クリック にとどまる。一方GA4では同ページが回遊先として機能している。

仮説: タイトル・h1・本文に 想定キーワードが不足している。例えば /gallery のタイトルが「Gallery | little seeds photography」のままだと、検索クエリと一致しない。

課題B: ランディング集中（88%がトップ）

検索ユーザーが gallery/voice/faq を直接見つけられていない。トップが落ちると全体が落ちる構造。

課題C: CV経路が脆弱

CV2件はすべて /contact 直来訪。plan→contact、gallery→contact の回遊CVがゼロ。各ページのCTA設計が弱い。

課題D: 商圏のミスマッチ

GA4の市区町村データを見ると、藤沢以外は質が低い: 横浜（量2位）は37秒、CV0。川崎・相模原は10秒程度。地域戦略は「広く浅く」より「藤沢で深く＋近接市の精鋭」が正解。

課題E: スタイル軸の正面勝負は不利

「ナチュラル」「自然体」KWは大手メディアが寡占。個人サイトが単独で取れる位置ではない。

4. 戦略の再設計（次の3ヶ月: 2026-05〜08）

方針
「広げる」から「深める」へ。ページを増やすフェーズは完了。次は 既存ページの密度・質を上げて、藤沢で2-3位を獲りに行く フェーズ。

⭐⭐⭐⭐⭐施策1: 「ニューボーンフォト 藤沢」で2-3位獲得

現状: 9.04位、CTR 17%　目標: 3位以内、月間クリック 100-150
根拠: baby marc以外は分散。CTRが既に高いので順位だけ上げれば良い

  • トップページに 撮影実績数（具体数値） を追加（baby marcは「8年以上」と明示）
  • トップに お客様の声の抜粋＋voiceページへのリンク を追加（信頼性シグナル）
  • トップに Instagram埋め込み を追加（鮮度シグナル、競合多数が実装済み）
  • トップh1/h2の 地域名密度を増やす（藤沢・湘南・茅ヶ崎・鎌倉の言及）
  • モバイル順位が3ランク下のため モバイル表示の改善（CLS/LCP実測、ヘッダー圧縮）

⭐⭐⭐⭐⭐施策2: MEO強化（GBP口コミ獲得）

現状: GBP登録済み・公開中、口コミ獲得施策なし　目標: 3ヶ月で口コミ10件、星評価4.8以上
根拠: 地域検索で マップパック表示の優先順位がオーガニックより上。藤沢市内ユーザーの「ニューボーンフォト 藤沢」検索でマップに表示されることが最強

  • 撮影完了後のサンクスメール／LINEに GBPレビュー誘導リンク を組み込む
  • GBPに 撮影投稿を週1更新（Instagram投稿の流用でOK）
  • GBPに 属性情報を充実（女性経営、出張対応、料金帯）

⭐⭐⭐⭐施策3: CV導線の強化（全ページ→contact）

現状: plan→contact、gallery→contact の回遊CVゼロ　目標: CV率 0.87% → 2%
根拠: PV総数は十分（trafficはある）、CV率を上げる方が新規流入獲得より速い

  • plan/gallery/voice の 各ページ末尾に固定CTA（「このプランで予約する」「料金を相談する」）
  • contact ページの 平均滞在12秒 = フォームが面倒・LINE導線が弱い可能性。LINE/問い合わせフォームの二択UIを目立たせる
  • voice/faq ページから LINEお問い合わせCTA をフローティングで設置

⭐⭐⭐⭐施策4: 商圏の再定義 — 藤沢/茅ヶ崎/鎌倉に絞る

現状: 横浜・湘南・川崎・神奈川を漫然とターゲット　目標: 藤沢を主、茅ヶ崎・鎌倉を副ターゲットに固定
根拠: GA4市区町村データで質が高いのはこの3市。湘南・鎌倉はライフパレット寡占で正面突破不可

  • 地域特化LPを2本だけ新設: /area/chigasaki/、/area/kamakura/
    • 各ページに「茅ヶ崎/鎌倉での撮影事例3件以上」「アクセス情報」「地域固有の撮影スポット紹介」を含める
    • 湘南LP・横浜LPは作らない（リソース集中）
  • 既存トップに「対応エリア: 藤沢・茅ヶ崎・鎌倉を中心に」と明記
  • BreadcrumbList 構造化データを追加

⭐⭐⭐施策5: スタイル軸はブログ記事型ロングテールで

現状: 「ナチュラル」「自然体」単独KWは大手寡占で取れない　目標: 複合長尾で月間表示 100-300
根拠: 「ナチュラル ニューボーンフォト 服装」「ニューボーンフォト 自然光 自宅」等の複合は大手も取りこぼしている

  • ガイドページ /guide/ を コラム集約ページ として新設
  • 記事候補:
    • ナチュラルなニューボーンフォトに最適な時間帯
    • 自然体の写真を撮るための自宅環境準備
    • ニューボーンフォト 服装の選び方（赤ちゃん・家族）
    • ニューボーンフォトの撮影時期と予約タイミング
  • 各記事3,000字以上、独自写真を3-5枚以上含める
  • 既存FAQと内部リンク連携

⭐⭐⭐⭐施策8: 検索結果のサムネ画像表示（マルチイメージSERP）対応

現状: 競合 OurPhoto は「神奈川県藤沢市でニューボーンフォト」の検索結果に 3枚の作品サムネが並ぶリッチリザルトを獲得。little seeds はサムネ表示なし　目標: 主要クエリ（藤沢／茅ヶ崎／鎌倉×ニューボーンフォト）で作品サムネ付き表示を獲得し、視覚優位のサービスとして CTR をさらに伸ばす
根拠: 写真サービスは「画像がSERPに出るか」でクリック率が大きく変わる。CTRが既に 17-18% と高い土台があるため、サムネ表示が加われば視認性で競合を上回れる。OurPhoto はタイトルに「神奈川県藤沢市でニューボーンフォト」と地域＋サービスを明示し、画像構造化と組み合わせて獲得している

  • 画像サイトマップの追加: sitemap.xml に image:image 拡張で、各ページが保持する画像URLとキャプションを明示。Googleに「このページにこの画像がある」と直接通知する
    • 例: <url><loc>.../gallery/</loc><image:image><image:loc>.../newborn-01.webp</image:loc><image:caption>藤沢の新生児フォト おくるみポーズ</image:caption></image:image></url>
  • 構造化データ（JSON-LD）の整備
    • トップ: LocalBusiness ＋ image（代表画像3-5枚を image 配列で指定）
    • /gallery: ImageGallery もしくは個別 ImageObject の集合
    • 各画像に contentUrl / caption / creator / datePublished を付与
  • alt属性の最適化: 全画像のaltを「藤沢 ニューボーンフォト おくるみポーズ」のように 地名＋サービス語＋シーン の自然文に書き直す（現状のロゴ/装飾用以外は全件対象）
  • figure / figcaption の活用: 主要作品画像を <figure> ＋ <figcaption> で構造化し、画像と説明文を意味的に結合する
  • og:image の個別化: 全ページ共通のロゴ画像をやめ、各ページの内容に応じた代表画像（gallery なら作品、plan なら撮影シーン）に差し替え
  • title／h1への地域＋サービス語の埋め込み: 施策6のタイトル見直しと連動。OurPhotoが「神奈川県藤沢市でニューボーンフォトの出張撮影」とtitleに明示している点が画像SERP獲得の前提条件
