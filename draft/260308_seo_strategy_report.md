## 3. 競合分析

### 検索上位サイトの傾向

「ニューボーンフォト 藤沢」の検索上位は以下の3カテゴリで構成されている。

#### (1) 大手マッチングプラットフォーム

- fotowa、OurPhoto、くらしのマーケット
- ドメインパワーが圧倒的に強く、上位を占有しやすい
- 個人サイトがこれらと正面から争うのは難しい

#### (2) 個人フォトグラファー / 専門スタジオ

- baby marc、ライフパレットフォトグラフィ、KUKKA、ベイビーズブレス
- 「藤沢」「湘南」に特化したコンテンツを持ち、上位表示に成功している
- **→ little seeds photography もこのカテゴリで十分戦える**

#### (3) 大手チェーン

- スタジオアリス、ハピリィ、たまひよ
- 藤沢エリアにも進出しているが、出張撮影特化ではないため差別化は可能

### キーワード難易度

**総合難易度: 中〜低**

- 大手プラットフォームに正面から勝つのは困難
- ただし個人フォトグラファーが上位に入っている実績があり、専門性の高いサイトなら上位表示は十分可能

## 4. 戦略の妥当性評価

### 結論：妥当だが、単一キーワードでは不十分

#### ポジティブ要因

- 「ニューボーンフォト 藤沢」は検索意図が明確で成約率が高い
- 個人サイトでも上位表示している競合がおり、勝てる余地がある
- 市場自体が成長中で、今後も検索需要の増加が期待できる

#### リスク要因

- 月間10〜100回の検索ボリュームは少なく、これだけではサイト流入が限定的（ただし+900%の急成長中）
- 大手プラットフォームがドメインパワーで上位を占有
- 現在3ページのみの小規模サイトでは、Googleからのコンテンツ評価が低くなりがち

#### 推奨アプローチ

「ニューボーンフォト 藤沢」だけでなく、**藤沢/湘南/鎌倉/茅ヶ崎/神奈川の地域クラスタ**に加え、**スタイル訴求キーワード（ナチュラル・自然体・シンプル）**もターゲットすることで、合算月間150〜1,500の流入を狙う。

## 5. 現サイトのSEO対策状況

### 良好な点

- title / description にキーワード（ニューボーンフォト、藤沢）が適切に配置されている
- 見出し構造（h1/h2/h3）が正しい階層で設計されている
- 全画像にalt属性が設定済み
- Google Tag Manager（GTM）が実装済み
- OGPタグ（トップページ）が設定済み
- `lang="ja"` 属性が設定済み

## 6. 上位表示に向けた施策案

### Phase 1: テクニカルSEO（即効性あり・コードで対応可能）

1. ✅ JSON-LD構造化データの追加（LocalBusiness, WebSite）
2. ✅ sitemap.xml の作成・Search Console登録
3. ✅ robots.txt の作成（Sitemap指定含む）
4. canonical タグの追加
5. OGP URL の https 化

### Phase 2: コンテンツSEO（中期）

1. Googleビジネスプロフィールの作成・最適化（MEO）
2. 撮影事例 / ギャラリーページの追加
3. お客様の声ページの追加
4. ブログ / コラムの追加（「ニューボーンフォトとは」「撮影時期」等）

### Phase 3: 外部施策（長期）

1. ✅ Google Search Console への登録・サイトマップ送信
2. SNS（Instagram）からの被リンク強化
3. 地域ポータルサイトへの掲載
