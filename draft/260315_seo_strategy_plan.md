# little seeds photography SEO戦略プラン

作成日: 2026-03-15

## Context

Search Console / GA4 の直近8日間データ（3/7〜3/14）と現サイトの分析から、以下の課題が明確になった。

**現状の数値**:

- 検索流入はわずか14.8%（16セッション/108セッション中）。主力はSNS流入43.5%
- 「ニューボーンフォト 藤沢」で掲載順位8.1位（モバイル9.3位）、クリック2回/週
- コンバージョン率 約1%（108セッション中、問い合わせ完了1件）
- サイトは実質2ページ（トップ + 問い合わせ）でコンテンツが圧倒的に不足

**テクニカルSEO基盤**: JSON-LD、canonical、OGP、sitemap、robots.txt、WebP最適化、クリティカルCSS等は実装済みで優秀。**ボトルネックはコンテンツの薄さと検索流入の弱さ**。

## キーワード戦略: 二軸アプローチ

「ニューボーンフォト 藤沢」単独ではボリュームが10〜100/月と少なく、1位を取っても月間20〜30クリックが上限。**地域軸とスタイル軸の二軸**でキーワードを攻める。

### 軸1: 地域キーワード（成約率が高い）


| キーワード         | 月間ボリューム          | 競合性 | 役割       |
| ------------- | ---------------- | --- | -------- |
| ニューボーンフォト 藤沢  | 10〜100（+900%成長中） | 中   | メイン地域ワード |
| ニューボーンフォト 湘南  | 10〜100           | 低   | 並行ターゲット  |
| ニューボーンフォト 鎌倉  | 10〜100           | 中   | 並行ターゲット  |
| ニューボーンフォト 茅ヶ崎 | 10〜100           | 中   | 並行ターゲット  |
| ニューボーンフォト 神奈川 | 10〜100           | 中   | 中間ワード    |


地域クラスタ合算: **月間50〜500**

### 軸2: スタイルキーワード（ボリュームが大きく競合性が低い）


| キーワード           | 月間ボリューム   | 競合性   | 役割           |
| --------------- | --------- | ----- | ------------ |
| ナチュラル ニューボーンフォト | 100〜1,000 | **低** | サイトの世界観と完全一致 |
| ニューボーンフォト 自然体   | 100〜1,000 | **低** | サイトの世界観と完全一致 |
| ニューボーンフォト シンプル  | 10〜100    | 低     | コンセプトとの親和性   |


スタイル系合算: **月間200〜2,000**

### なぜ二軸か

- 地域ワードだけでは検索ボリュームの天井が低い
- スタイル系は**ボリュームが地域系の10倍**かつ**競合性が低い**
- サイトのコンセプト「ナチュラル＆シンプル」と完全に一致しており、自然なキーワード散布が可能
- 地域ワード = 「どこで撮るか」を決めたユーザー、スタイルワード = 「どんな写真が撮りたいか」を探しているユーザー。両方をカバーすることでファネルが広がる

## Phase 1: 短期施策（1〜2週間）— 既存ページの最適化

### 1-1. トップページのテキスト拡充（優先度: 高）

**対象**: `web/index.html`


| セクション         | 現状         | 改善内容                                                                      | 状態    |
| ------------- | ---------- | ------------------------------------------------------------------------- | ----- |
| hero subtitle | 「藤沢市を拠点に〜」 | 「神奈川県藤沢市を拠点に、湘南・鎌倉・茅ヶ崎エリアへ〜」に地域名追加                                        | ✅ 済   |
| Concept       | 短い詩的テキストのみ | ニューボーンフォトの簡単な説明（撮影時期・出張撮影のメリット）を追加。**「ナチュラル」「自然体」「シンプル」等のスタイルワードを自然に含める** | ⬜ 未着手 |
| Profile       | 1文のみ       | 撮影実績数・保有資格・藤沢市在住の明示を追加                                                    | ✅ 済   |
| Area          | 料金表のみ      | 「藤沢市内は出張費無料」等の補足テキスト追加                                                    | ⬜ 未着手 |


**キーワード散布の方針**:

- 地域軸: 藤沢、湘南、鎌倉、茅ヶ崎、神奈川を各セクションで自然に言及
- スタイル軸: ナチュラル、自然体、シンプル、出張撮影をConceptセクション中心に散布
- 不自然なキーワード詰め込みは厳禁。あくまでサイトの世界観に合った自然な文脈で

**効果**: 地域キーワード + スタイルキーワードの二軸で検索表示機会を拡大。「ニューボーンフォト 藤沢」の順位改善に加え、「ナチュラル ニューボーンフォト」（100〜1,000/月）での表示開始を狙う

## Phase 2: 中期施策（1〜3ヶ月）— コンテンツ拡充

### 2-1. ギャラリーページ新設（優先度: 最高）

```
web/gallery/index.html
```

- 撮影事例を写真+キャプション（エリア・月齢・プラン名）で掲載

- ImageGallery 構造化データ

**効果**: コンバージョン率改善（1% → 2〜3%）、画像検索流入、滞在時間向上

### 2-2. お客様の声ページ新設（優先度: 高）

```
web/voice/index.html
```

- Review 構造化データ（星評価のリッチリザルト可能性）
- 各レビューに撮影エリア・プラン・時期を付記
- 「口コミ」「評判」系キーワード獲得

### 2-3. ニューボーンフォトガイドページ新設（優先度: 高）

```
web/guide/index.html
```

- title: `ニューボーンフォトとは？撮影時期・準備・料金を藤沢のフォトグラファーが解説`
- 情報クエリ（「ニューボーンフォトとは」「いつ」「準備」「料金相場」）を獲得
- **「ナチュラルニューボーンフォトとは」「自然体の撮影スタイル」等のスタイル訴求セクションを含める**
- ファネル上流（情報収集段階）のユーザーを捕捉し、内部リンクでコンバージョンへ導く

### 2-4. 地域特化ページ新設（優先度: 中 / 十分なコンテンツがある地域から）

```
web/area/fujisawa/index.html   — 藤沢市
web/area/shonan/index.html     — 湘南エリア
```

- 各地域固有の撮影事例・お客様の声・アクセス情報を含む（テンプレコピペ厳禁）
- 地域クラスタ（藤沢/湘南/鎌倉/茅ヶ崎/神奈川）合算で月間150〜1,500のボリュームをカバー

### 2-5. サイト全体の導線・内部リンク整備

- 共通ヘッダー/フッターにサイトナビゲーション追加
- 各ページにパンくずリスト + BreadcrumbList構造化データ
- コンテンツ内の相互リンク（ガイド→プラン、ギャラリー→声→問い合わせ）
- `web/sitemap.xml` に全新規ページ追加

## 検証方法

- Search Console で主要キーワード（「ニューボーンフォト 藤沢」「ナチュラル ニューボーンフォト」「ニューボーンフォト 自然体」「ニューボーンフォト 湘南」等）の順位・表示回数・CTRを週次で確認
- GA4 で検索流入比率・コンバージョン率・ページ別PVを週次で確認
- `tools/fetch_gsc.py` / `tools/fetch_ga4.py` でデータ取得し、施策前後の比較
- 新規ページ公開後、Search Console の「URL検査」でインデックス状況を確認
- リッチリザルトテストでFAQ構造化データの検証
