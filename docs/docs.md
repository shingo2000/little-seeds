# little seeds

## キャッチコピー

little seedsは「ちいさな種たち」の意味です。
種たちが芽を出し、根を張り、花を咲かせて実をつける。
ちいさな命の誕生を祝い、成長を応援したいという思いを込めてこの名前をつけました。


## コンセプト

何年経っても新生児の頃を思い出せるように。
無理のない安全なポージングと赤ちゃんの表情を大切に撮影しています。
赤ちゃんの無垢な美しさが引き立つようにナチュラル＆シンプルにコーディネート。
初めての記念撮影がご家族の素敵な思い出になるよう、心を込めて撮影させていただきます。

## 安心安全への取り組み

専門知識に基づいた撮影: 専門講座で新生児の安全なポージング、理学療法士による新生児の身体の基礎講座を学んでいます。
無理のないポージング: 赤ちゃんの体の負担を第一に考え、関節や骨の特性を考慮してポージングします。合成が必要な頬杖ポーズ、支えなく赤ちゃんが自立したポーズは行っておりません。過度に首や身体を曲げたり、赤ちゃんを高いところに寝かせて撮影しません。
衛生管理: 赤ちゃんの肌に触れる衣装や小物は、すべて撮影のたびに無添加洗剤で洗濯、個別の袋に入れ持参します。素材や肌触りなどにも配慮しています。
撮影時の温度管理　新生児は体温調節機能が未熟なため、撮影時は衣装に合わせた室温管理を行います。赤ちゃんが心地よく過ごせる環境を整えます。
1日1組限定: 1日1組、1ヶ月5組までの予約制。余裕を持ってスケジューリングできるようにしています。産後のお問合せにも可能な限り対応します。
キャンセル・延期無料: 赤ちゃんやご家族、またフォトグラファーの体調不良による日程変更は無料で承ります。

1. 仮予約（妊娠６ヶ月〜）: プランが決まりましたらホームページの予約フォーム、又は公式LINEよりご予約ください。出産予定日を仮予約にします。出張費のお見積もり、撮影スペースの確認などを事前に行います。
2. 出産報告: ご出産後、なるべく早めにご連絡下さい。赤ちゃんとママの体調を考慮し撮影日を決定します。生後7日〜14日に調整する場合が多いです。撮影当日の細かなご案内をお送りします。
3. 撮影当日: ご自宅へ伺い、赤ちゃんの様子に合わせて撮影します。撮影代金は当日現金にてお支払い、または後日お振込みをお願いします。
4. 納品: お振込確認後撮影から3週間以内に、ダウンロード形式にてデータを納品します。パネル等のプリント商品は写真のセレクト後1ヶ月程度でお届けします。
